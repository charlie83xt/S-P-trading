
# Ironbeam Key Findings

Ironbeam offers a fully-featured futures trading API (Ironbeam xAPI) with:
*   **Real-time data and trade execution:** Hosted directly at the exchange data center for low latency.
*   **Easy Integration:** REST API with documentation in popular programming languages.
*   **Unfiltered Market Data:** Provides unfiltered Level 2 market data via WebSockets.
*   **Account Management & Risk Management:** Features for monitoring positions, balances, order history, and implementing risk parameters.
