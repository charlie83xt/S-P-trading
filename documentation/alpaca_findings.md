# Alpaca Key Findings

Alpaca offers APIs for stock, options, and crypto trading. 

**Key Features for this project:**
*   **Testing Environment:** Alpaca provides a testing environment, which is essential for developing and testing trading strategies without risking real capital.
*   **SDKs:** They offer SDKs in multiple languages, including Python, which is a suitable language for algorithmic trading.
*   **Commission-Free API:** Their API can facilitate commission-free stock trading.

**Next Steps:**
*   Investigate their documentation further to confirm futures trading support and historical data access.
*   Compare with other platforms.




**Futures Trading Support:**
Alpaca's documentation states: "Stay tuned for our API updates as we have on roadmap plans for futures, FX, and much more!" This indicates that futures trading is not currently supported, but is planned for the future.

**Historical Data:**
Alpaca provides access to "real-time market pricing data and up to 6+ years worth of historical data for stocks and crypto." While this is good for stocks and crypto, it doesn't explicitly mention futures data.

**Conclusion for Alpaca:**
Alpaca is a strong candidate for stock and crypto trading due to its testing environment and Python SDK. However, it does not currently support futures trading, which is a core requirement for this project. It might be a good option for testing general trading bot logic, but not for futures-specific strategies.
